# Embedded_Operating_System_PTIT_Exercise_5
Mục tiêu cốt lõi của bài tập này là hiểu và làm chủ được quy trình phát triển phần mềm cho hệ thống nhúng Linux (cụ thể là board BeagleBone Black kiến trúc ARM) thông qua công cụ Buildroot. Quy trình này bao gồm: sử dụng thư viện nguồn mở có sẵn, tự phát triển thư viện cá nhân, so sánh các cơ chế liên kết (Linking), và cuối cùng là đóng gói ứng dụng thành một module hệ thống.
